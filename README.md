# Hello World Android

Simple Android application written in Kotlin and built with Gradle.

## Requirements

- Android Studio
- Android SDK 35
- JDK 17 or newer

## Run

1. Clone the repository:

   ```bash
   git clone https://github.com/hr-al/Test.git
   cd Test
   ```

2. Open the project in Android Studio.
3. Let Android Studio sync Gradle.
4. Run the `app` configuration on an emulator or Android device.

The app displays `Hello World` on the screen.
